# Shortcodes

Shortcodes are dynamically generated text, written in the format `[shortcode-name]`.
They can be used in post body.

## Insert Post

Insert a post by #id.

```
[post #11]
```

## Calendar

Display a month calendar with marked dates.

```
[calendar year=yyyy month=mm mark=16]
```

## Poll

Display a polling box, with given options. Team members can vote on one of these options.

Note: Do not change option numbers after the poll starts.

```
[poll name=colors option1=Red option2=Green option4=Blue]
```
